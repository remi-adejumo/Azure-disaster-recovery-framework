# Azure Disaster Recovery Framework (Enterprise)

This repository contains an enterprise-grade Disaster Recovery (DR) framework aligned to the Azure Well-Architected Framework.

It includes reference architecture, DR/BCP tiering, runbooks, cost models, Terraform templates, and operational guidance for implementing resilient cloud platforms.
